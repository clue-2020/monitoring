МЕТОДИЧЕСКИЙ ИНСТРУМЕНТАРИЙ КОНЪЮНКТУРНОГО ОБСЛЕДОВАНИЯ ДЕЛОВОЙ АКТИВНОСТИ И НАЛОГОВОГО СТИМУЛИРОВАНИЯ СУБЪЕКТОВ ИННОВАЦИОННОЙ ДЕЯТЕЛЬНОСТИ (ПЛАН А, ЭЛЕКТРОННАЯ ФОРМА А)

1) Участники мониторинга формируют постоянно действующую респондентскую интернет-сеть для непрерывного обмена информацией и знаниями.

2) Электронная форма А состоит из 2 разделов: А1 (деловая активность)  и А2 (налоговые стимулы и результаты). Каждый раздел включает определенное количество индикаторов (F). Индикаторы раздела А1: (F1-конкурентоспособность); (F2-занятость); (F3-денежные ресурсы); (F4-использование прибыли); (F5-бизнес-актиивы); (F6-конъюнктура); (F7-технологические инновации). Индикаторы раздела А2: (F8-вовлеченность в режим налоговых льгот); (F9-отдача от вовлеченности в режим налоговых льгот). Каждый индикатор содержит определенный перечень вопросов (n). 

3) Частная позиция информанта на каждый вопрос (n) электронной анкеты может быть: оптимистичной (1 голос), нейтральной (0,366 голоса), пессимистичной (0,134 голоса), уровень полной неопределенности или отсутствия оснований для частной позиции (0 голосов). 

4) Важнейшими метриками конъюнктурного обследования деловой активности и налогового стимулирования инновационной деятельности организации являются: 

Коэффициент делового оптимизма (КДО) и коэффициент налогового оптимизма (КНО), определяемый для каждого F по формуле: сумма частных позиций информанта, деленная на максимально благоприятный результат для исключительно оптимистических позиций информанта для всех (n). Значение КДО и КНО находится в интервале [0-1].   

Коэффициент деловой активности (КДА) и коэффициент налогового стимулирования (КНС), определяемый для каждого F по формуле: сумма частных позиций информанта, деленная на максимально благоприятный результат из лучших оценок участников сети или ее целевого сегмента (закона возможного сети или ее целевого сегмента, например, целевым сегментом сети являются организации Белнефтехима). Значение КДА и КНС находится в интервале [0-1].  

Коэффициент ожидаемой деловой активности (КОДА) является произведением КДО на КДА  для каждого F1-F7 и их среднего значения, т.е. КОДА=КДО*КДА и соответственно коэффициент ожидаемого налогового стимулирования (КОНС) является произведением КНО на КНС для каждого F8-F9 и их среднего значения, т.е. КОНС=КНО*КНС.  

Максимально благоприятный результат стремится к 1.

Анкета содержит также показатели научной, научно-технической и инновационной деятельности организации, которые можно отслеживать в динамике.

Коэффициент делового оптимизма (КДО) определяемый для F1, F2, F3, F4 соответственно по формуле: сумма частных позиций информанта, деленная на максимально благоприятный результат для исключительно оптимистических позиций информанта для всех (n). Значение КДО находится в интервале [0-1].

Коэффициент деловой активности (КДА) определяемый для F1, F2, F3, F4 соответственно по формуле: сумма частных позиций информанта, деленная на максимально благоприятный результат из лучших оценок участников сети или ее целевого сегмента (закона возможного сети или ее целевого сегмента, например, целевым сегментом сети являются организации Белнефтехима). Значение КДА находится в интервале [0-1].  

Коэффициент развития инновационной системы (КРИС) является произведением КДО на КДА для каждого F1-F4 и их среднего значения, т.е. КРИС=КДО*КДА 

Максимально благоприятный результат стремится к 1.
