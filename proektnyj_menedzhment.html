ЭЛЕКТРОННАЯ ФОРМА (ДЕМОВЕРСИЯ)

Идентификация приоритетов, возможностей и серьезности намерений организации, претендующей на реализацию инновационного (инвестиционного) проекта

1. Электронная заявка является основанием для включения организации в лонг-лист (предварительный список) участников бизнес-планирования (резидентской сети). Право на получение налоговых льгот имеют организации, успешно прошедшие этап электронной регистрации.
2. Электронная заявка организации состоит из 3 сценарных планов: F1 - приоритеты; F2 - возможности; F3 - действия, связанные с реализацией проекта. Каждый сценарный план (атрибут) включает определенное количество индикаторов (N): F1 (N) - 4 индикатора; F2 (N) - 9 индикаторов; F3 (N) - 4 индикатора. Каждый индикатор содержит перечень вопросов (n).
3. Заявленная позиция организации учитывается по каждому вопросу (n) и имеет 4 уровня значимости: оптимистическая позиция (1 голос), нейтральная позиция (0,366 голоса), пессимистическая позиция (0,134 голоса), уровень полной неопределенности или отсутствия частной позиции (0 голосов) .
4. Для принятия решений по сценарному плану (атрибуту) рекомендуется использовать критерий умеренного оптимизма Гурвица: H (F) = max F (N) * c + (1- c) * min F (N), где с = 0, 5.
5. Коэффициент вовлеченности организации в научную, научно-техническую, инновационную деятельность определяет по формуле:
K (F1) = F1 (N) max F1 (N), где K (F1) - коэффициент вовлеченности организации в научную, научно-техническую, инновационную деятельность по атрибуту F1 (N); max F1 (N) - значение, полученное из лучших оценок участников резидентской сети.
K (F2) = F2 (N) max F2 (N), где K (F2) - коэффициент вовлеченности организации в научную, научно-техническую, инновационную деятельность по атрибуту F2 (N); max F2 (N) - значение, полученное из лучших оценок участников резидентской сети.
K (F3) = F3 (N) max F3 (N), где K (F3) - коэффициент вовлеченности в научную, научно-техническую, инновационную деятельность по атрибуту F3 (N); max F3 (N) - значение, полученное из лучших оценок участников резидентской сети.
6. Организация может рассчитывать на участие в резидентской сети бизнес-планирования, если средневзвешенный результат принимаемого решения комбинацией аддитивного и мультипликативного метода построения обобщенного критерия K (F)> = 1:
К (F) = c * ((K (F1) + К (F2) + K (F3)) + (1-c) * K (F1) * К (F2) * K (F3)> = 1, где с = 0,5
